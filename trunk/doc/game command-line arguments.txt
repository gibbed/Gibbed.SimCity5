-f

  Run game in fullscreen mode.


-jobPriority:<value>

  Job thread priority. Only used if -noBackground is not specified.


-jobThreads:<value>

  Number of job threads. Only used if -noBackground is not specified.
  When not specified, default value is number of processors minus one.
  Maximum specifiable value is 20.


-noBackground

  Disables background threads.


-nomute

  Prevents muting of audio when the game is not focused.


-noMP

  Disables setting of process affinity mask.


-noVSync

  Disables vsync.


-state:<value>

  Set initial game state, ie try -state:SimCity.


-vSync

  Enables vsync.

-w

  Run game in windowed mode.
